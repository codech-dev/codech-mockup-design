# Codech Mockup Design Plugin

This plugin provides the `codech-mockup-design` skill for structured design ideation and visual prototyping.

## Available Skills

- **codech-mockup-design** — Structured 7-phase workflow that produces design system documents and self-contained HTML showcases. Triggers when users ask to design UI, create mockups, prototype screens, explore visual directions, or build design systems.

## Dependencies

- **`ui-ux-pro-max` skill** — Required for design intelligence queries (color palettes, font pairings, UX patterns). Install separately.
- **Python 3** — Required by ui-ux-pro-max's search scripts.
- **Internet connection** — For Tailwind CDN and Google Fonts in showcase HTML files.
